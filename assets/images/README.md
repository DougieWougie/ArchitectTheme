# Images Directory

## Required Image

Please add your profile picture to this directory:

**File:** `ProfilePicture.png`
- **Size:** 200x200px or larger (recommended 400x400px for retina displays)
- **Format:** PNG (preferred) or JPG
- **Shape:** Should be square (will be displayed as a circle)

The profile picture will appear in the header at 200px and will spin 360° when hovered over.

### Responsive Sizes:
- Desktop: 200x200px
- Tablet/Mobile: 120x120px
- Small screens: 80x80px
- Landscape mode: 60x60px
